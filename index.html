LETAL QUIZ

Bienvenid@s a nuestro super juego de preguntas. Si estás preparado/a para enfrentarte a este reto, sólo tienes que leerte unas sencillas normas y pulsar el botón Empezar. Que la suerte te acompañe!!

Lee y recuerda las siguientes normas, una vez comenzado el juego ya no hay vuelta atrás. Trata de acertar el mayor número de respuestas posibles.

  1. Presta atención al leer las preguntas y las respuestas.
  2. Una vez hayas elegido la respuesta correcta, ésta se almacena y no se puede cambiar.
  3. Cada pregunta sólo tiene una respuesta correcta.
  4. No puedes abandonar el juego hasta que no hayas contestado a todas las preguntas.
  5. Una vez terminado, pulsa el botón Submit para ver los resultados.
  6. El primer desafío consiste en pulsar un botón para empezar el juego.
